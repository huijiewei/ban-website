### 只对 github.com 开启代理

socks5h:// 中的 "h" 意思是在代理端使用远程 DNS，而不是你的本地 DNS。（懂得都懂, 刚才一直推送不上代码，烦）

```bash
git config --global http.https://github.com.proxy socks5h://127.0.0.1:1080
```

### Git 使用 macOS 钥匙串保存密码

```bash
git config --global credential.helper osxkeychain
```
